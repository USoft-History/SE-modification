## RU

Архив старых исходников, утилит, патчей и драйверов для кастомизации и прошивки легендарных телефонов **Sony Ericsson** (серии W, K и др.). Собранные материалы эпохи расцвета патчестроительства и моддинга мобильных устройств.

---

### 📂 Структура репозитория

* **`m600/Драйвера`** — Драйверы и софт для UIQ-смартфона Sony Ericsson M600.
* **`w200`** / **`w610`** / **`w800`** — Файлы, специфичные для конкретных моделей Walkman (патчи, бэкапы, драйверы дисплеев/камер, эльфы).
* **`Патчестроительство`** — Скрипты, ассемблерные исходники (`.asm`), конфиги и утилиты для создания VKP-патчей.
* **`Программы`** — Классический софт для работы с прошивками (FS/MAIN): *Setool2lite, FAR Manager + SEFP/JustDaFlash, XS++, GBlocker, VKP-патчеры* и др.
* **`Прошивочные драйвера`** — USB-драйвера (USB Flash Drivers), необходимые для перевода телефона в режим прошивки (зажатие кнопки `C` или `2+5` при подключении кабеля).
* **`Финализации`** — Файлы кустомизации (CDA) и скрипты финализации для разных регионов, необходимые после заливки основной прошивки.
* **`Прочее`** — Различные полезные материалы, мануалы и неразортированные сопутствующие файлы.

---

### 🛠️ Инструменты из прошлого

Для работы с файлами из этого архива (если вы решите оживить старый девайс) обычно использовались:
* **FAR Manager** со специальными плагинами для доступа к файловой системе телефона.
* Кабель **DCU-60** (или аналогичный).
* Windows XP / 7 (для корректной работы старых прошивочных драйверов может потребоваться виртуальная машина или отключение проверки цифровой подписи).

## EN

An archive of old source code, utilities, patches, and drivers for customizing and flashing legendary **Sony Ericsson** phones (W-series, K-series, etc.). Collected materials from the golden era of patch-making and mobile device modding.

---

### 📂 Repository Structure

* **`m600/Драйвера`** — Drivers and software for the Sony Ericsson M600 UIQ smartphone.
* **`w200`** / **`w610`** / **`w800`** — Model-specific files for Walkman phones (patches, backups, display/camera drivers, ELF applications).
* **`Патчестроительство`** (Patch-making) — Scripts, assembly source files (`.asm`), configs, and utilities for creating VKP patches.
* **`Программы`** (Software) — Classic firmware tools (FS/MAIN): *Setool2lite, FAR Manager + SEFP/JustDaFlash, XS++, GBlocker, VKP patchers*, etc.
* **`Прошивочные драйвера`** (Flashing Drivers) — USB Flash Drivers required to put the phone into flashing mode (holding the `C` or `2+5` key while connecting the cable).
* **`Финализации`** (Finalizations) — Customization files (CDA) and finalization scripts for different regions, required after flashing the main firmware.
* **`Прочее`** (Miscellaneous) — Various useful materials, manuals, and unsorted accompanying files.

---

### 🛠️ Tools from the Past

To work with the files in this archive (should you decide to revive an old device), the following tools were traditionally used:
* **FAR Manager** with specialized plugins to access the phone's file system.
* **DCU-60** cable (or equivalent).
* Windows XP / 7 (a virtual machine or disabling driver signature enforcement may be required for old flashing drivers to function correctly).
