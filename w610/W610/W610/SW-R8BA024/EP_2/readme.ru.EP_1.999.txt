В данном каталоге содержатся файлы необходимые для установки ЭП 1.999

1) Создаём необходимую структуру каталогов для размещения эльфов
В памяти телефона:
usb/other/ZBin
usb/other/ini
usb/other/ZBin/Config
usb/other/ZBin/DLL

На карте памяти:
/other/ZBin/Daemons
/other/ZBin/Config

2) Копируем необходимые файлы компонентов ЭП
Скопируем файл констант DYN_CONST.bin в usb/other/ZBin/Config
Скопируйте файл LibraryDLL.dll и прочие dll, если такие имеются в наличии в каталог usb/other/ZBin/DLL/

3) Устанавливаем два* vkp-патча:
- эльфлоадер XXXX_YYYYYYY.vkp
- библиотеку функций XXXX_YYYYYYY.lib.vkp
Прим.: для А2 эльфлоадар может состоять из двух частей

Для А2, следует учесть:
DB3150: патчи EP, Lib заливать в phone_app.cxc

DB3200/3210: 
Одна "часть" эльфлоадера устанавливается в phone_app.cxc, а друга в phone_emp_app.cxc 
Библиотека функций Lib заливать в phone_app.cxc

FAQ:
Q: для чего тут выложено lib.pkg
A: Этот архив содержит библиотеку функций, константы и DLL файлы необходимые для обновления ЭП.
Для установки, при правильно настроенной ассоциации эльфов достаточно распаковать его,
файлы будут автоматически размещены в нужных местах. Или руками распаковать и разложить всё по местам.
Также используется для работы LibUpdater, ява-приложения которое позволяет загружать этот файл с нашего сервера.
